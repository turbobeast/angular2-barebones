# angular2-barebones

## Getting Started

### Clone the Repo and install depenedencies

```bash
git clone git@github.com:turbobeast/angular2-barebones.git
cd angular2-barebones
yarn install
typings install
```

### Run Unit Tests

```bash
yarn global add karma-cli
npm test
```

### Build
```bash
npm run build
# or
npm run build:prod
```

### Aot

```bash
npm run build:aot
```
